Para a instalação do Bootsrtrap pip:
curl https://bootstrap.pypa.io/get-pip.py -o get-pip.py

----------------------------------------------------------------------------------------------------------

Para criar o ambiente virtual execute no prompt de comandos no terminal aberto na pasta de seu projeto:

python -m venv venv

.\venv\Scripts\activate

----------------------------------------------------------------------------------------------------------

Resumo do fluxo de trabalho
Criar: python -m venv venv

Ativar: .\venv\Scripts\activate

Instalar: pip install django

Trabalhar: Começar seu projeto Django!

Sair da venv: deactivate 

----------------------------------------------------------------------------------------------------------

Para amnater atualizado sempre rodar o:

pip freeze > requirements.txt

----------------------------------------------------------------------------------------------------------

Para upar no git

git remote add origin git@github.com:Scalabrini17/alura_space.git
git branch -M main
git push -u origin main

Se precisar de uma ajudinha git no terminal 

1. Alterar o ÚLTIMO commit (Para mudar apenas a mensagem do commit)
git commit --amend -m "Sua nova mensagem correta aqui"

2 Para adicionar um arquivo que você esqueceu:,
git add nome_do_arquivo
Atualize o último commit sem alterar a mensagem original:
git commit --amend --no-edit
